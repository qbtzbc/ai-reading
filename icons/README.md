# 图标文件说明

此目录应包含以下图标文件：
- icon16.png (16x16 像素)
- icon32.png (32x32 像素)  
- icon48.png (48x48 像素)
- icon128.png (128x128 像素)

临时解决方案：可以使用任何16x16、32x32、48x48、128x128的PNG图标文件，
将它们重命名为对应的文件名放在此目录中。

或者使用在线图标生成工具：
1. 访问 https://www.favicon-generator.org/
2. 上传一个图片或创建一个简单图标
3. 下载生成的不同尺寸图标文件
4. 重命名并放置在此目录
